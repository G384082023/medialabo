各授業回の学習事項

第1回 WWW と HTML

学習事項:

初回はHTMLの書き方を中心に話す。 最も重要な点はpと/pで最初と最後を囲んだ一部の要素になる。
たくさんの要素がある。 例えば、HTMLでの空白の書き方、HTMLでの特殊なテキストの書き方、HTML文書のコメントの追加方法など。
ハイパーリンクを設定する時は、それぞれの href属性、title属性と target属性を利用する。

演習の成果： 演習1-4:HTMLのさまざまな要素

第2回 CSS による Web ページの装飾

学習事項:

第二回の授業の重要なことは、CSS を構築して HTML ファイルに関連付ける方法。
CSS ファイルを使用すると、HTML ファイル内で変更を加えることなく、CSS で変更できる。
たとえば、h1 のテキストの色を変更したい場合は、CSSを使用するだけでいい。
もう一つ重要な点はレイアウト。
レイアウトはフレックスボックスとグリッドレイアウトがある。
フレックスボックスは複数の要素を並べたもので、それぞれの要素の幅を薄くするのは手间がかかるもの。
グリッドレイアウトは、要素を行と列にエリアカットしてその真ん中にサブ要素を構成するレイアウ。
これらを通じて、さまざまな部分内の物事を区別し、それらを背景色で塗りつぶすことができる。

演習の成果： ウェブブラウザ説明ページの装飾
